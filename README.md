### In plain English describe the function and use of a multiplexer.
    The function and use of a mux is to be able to choose from n amount of inputs and go into one output, this allows you to select one of the inputs using the signal lines which theres log_2 n amount of signal wires, can be used as a compressor.
### In plain English describe the function and use of a demultiplexer.
A demultiplexer is the exact opposite of a multiplexer, it has n amount of outputs and also has log_2 n number of signal wires and one input, can be used as a expander.
